# Tourists dashboard

A work-in-progress dashboard for [tourists](https://github.com/amanda-amy-frost/tourists) hosted on my [website](https://amandafrost.eu/dashboard/).

Made in Python with [marimo](https://marimo.io/) notebooks and [Vega-Altair](https://altair-viz.github.io/) for data visualization. [uv](https://docs.astral.sh/uv/) also makes dependency management a non-issue.

Be aware of slow initial load times. There is a lot of code that has to load on boot, but afterwards the experience is smooth due to running on [WebAssembly](https://webassembly.org/).
